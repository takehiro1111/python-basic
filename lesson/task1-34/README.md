# 要件
```txt
## 商品マスター登録プログラムを作成する

以下の仕様を満たすプログラムを作成してください。

### メインメニュー

ユーザーに以下の選択肢を提示し、対応する機能を実行するプログラムを作成します：

1. **商品一覧表示**
2. **商品登録**
3. **商品削除**
4. **商品CSV出力**
5. **終了**

---

### 各機能の詳細

### **1. 商品一覧表示**

登録されている商品のリストを以下の項目で表示します：

- **ID**
- **商品名**
- **JANコード**

---

### **2. 商品登録**

新しい商品を登録します。

- **商品名**: ユーザーから入力を受け付けます。
- **JANコード**: 自動生成されます。
    - 生成ルール: **9桁のランダムな数字** + **IDを3桁でゼロ埋めした番号**
    - 例: 商品のIDが1の場合、JANコードは `123456789001` のようになります。

---

### **3. 商品削除**

指定されたIDの商品を一覧から削除します。

- 削除する商品のIDをユーザーに入力させます。

---

### **4. 商品CSV出力**

現在登録されている商品一覧をCSVファイルとして出力します。

- 出力先: `./csv/item_list_{現在時刻}.csv`
    - 例: `./csv/item_list_20250113123045.csv`
- ファイルに含まれる項目:
    - **ID**
    - **商品名**
    - **JANコード**

---

### **5. 終了**

プログラムを終了します。

---

この仕様を基にプログラムを設計・実装してください。
```

# 処理の方向性

## 1,フロントになるメニュー選択
- どの処理を行うかフロントで入力処理を挟みながら選択する。

## 2.メイン処理の作成(商品一覧表示 / 商品登録 / 商品削除)
- メニュー選択とかではなくて、既存で登録済み商品の情報を取得
- データに永続性は持たせない。(プロセスごとの管理)


## 3.商品データの処理後に最新の状態をCSVファイルとして出力
- ファイルは日時で一位のものを作成する。
